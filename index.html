About My Bots

I'm building bots for WhatsApp and Discord. Right now, most of my focus is on the WhatsApp bot — using JavaScript, Baileys 7.0.0-rc.9, JSON systems and a full command handler all written by me.

The bot includes systems like Cards, Characters, Games, Economy, Moderation, Media Tools, Utilities, Anime Features, Yugi-Oh Cards, and more. Everything runs using # as the prefix, and you can check command info with #help <command>.

The Discord version is in early planning, but the goal is to match or surpass the WhatsApp features. I’m also learning HTML, CSS, JS, JSON, and Python as I build all this from scratch.
